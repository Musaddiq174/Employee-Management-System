Office Employee Management System

The Office Employee Management System is a Django-based web project that helps manage employee records easily. It allows users to add, view, remove, and filter employees using simple HTML templates rendered through Django views.

Features

View All Employees: Display all employees stored in the database.

Add Employee: Add a new employee with department, role, salary, and hire date.

Remove Employee: Delete existing employee records.

Filter Employee Details: Search and filter employees based on different criteria.

Tech Stack

Backend: Django (Python)

Frontend: HTML, CSS (Django Templates)

Database: SQLite (default Django database)
